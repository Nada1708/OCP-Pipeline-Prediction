# Prédiction du Débit Horaire — Pipeline OCP Khouribga → Jorf Lasfar

## Description
Modèle de Machine Learning développé durant mon stage à OCP Jorf Lasfar.

L'objectif est de prédire le débit horaire nécessaire à envoyer depuis 
Khouribga vers les clients de Jorf Lasfar, en tenant compte de l'heure 
de la journée et de l'historique des capteurs du pipeline.

## Résultat
- Modèle : XGBoost
- R² = 0.96
- Erreur moyenne : 14 t/h

## Technologies
Python · XGBoost · MLflow · Streamlit
